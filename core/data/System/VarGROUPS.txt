#VarGROUPS
---+++ GROUPS -- a formatted list of groups
*Deprecated - do not use. Use VarGROUPINFO instead*
   * Expands to a formatted list of user groups in your %WIKITOOLNAME%.
   * Syntax: =%<nop>GROUPS%=
   * The macro is intended to be used in %USERSWEB%.WikiGroups, to allow a
     group listing for various user mapping managers.
