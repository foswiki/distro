Introduction

If your Foswiki site is already working, continue to browse to the Foswiki WebHome.
You will now need to consider how you are going to manage authentication and access control. See the reference manual sections on authentication and access control, and the 'Security' section.

To become an administrator you need to first register as a normal user and then goto %ADMINGROUP% and add that user to the GROUP setting (follow the text in yellow).

If you are on a non-standard platform or environment there are a lot of supplemental documents on Foswiki.org

If you get stuck there is a lot of support available at the Support forum.

Explanation of colours and symbols:
  • Settings marked like this are required (they must have a value).
  • Errors in your configuration will be highlighted.
  • Warnings are non-fatal, but are often a good indicator that something that is wrong.
  • Expert settings are for expert use only. You should not fiddle with them unless you have read all the documentation first. Expert options are hidden unless you click the "Show EXPERT options" button
