---++ Usage

The icons are attached to the FamFamFamFlagIcons  FamFamFamMiniIcons  FamFamFamMintIcons  
FamFamFamSilkIcons FamFamFamSilkCompanion1Icons topics - 
be *CAREFUL* those topics can cause your browser to run very slowly for a bit.

At this point, the only way to access these Icons is using the url - see below.

---+++ Topic action icons.
To replace the _Edit_, _Attach_ topic action buttons with icons, pre-pend your =SKIN= setting with =famfamfam=.

eg
<verbatim>
   * Set SKIN=famfamfam,pattern
</verbatim>

---++ Work in progress: DocumentGraphics themeing
You can aso use %SYSTEMWEB%.FamFamFamGraphics as a replacement for the Foswiki DocumentGraphics, 
by setting =ICONTOPIC= on your %USERSWEB%.SitePreferences, and enabling the DocumentGraphicsPlugin.

<verbatim>
   * Set ICONTOPIC = %SYSTEMWEB%.FamFamFamGraphics
</verbatim>

---+++ Foswiki Extra Icons 
So far built by Sven based on the Silk ones above, except for 
   1 the XML icon - which comes from http://commons.wikimedia.org/wiki/File:XML_icon.svg
   2 the targets which are based on - http://www.favicon.cc/?action=icon&file_id=37384
   3 the Universal edit icons are based on the current (dec 2009) svg
   4 hand made done, closed, todo, updated icons - yes, ok, so i'm not an artist.
   5 external sites icons like skype are made using their favicons
