---++ Who should read this document?

Most html elements generated by Foswiki core code now have Cascading Style Sheet (CSS) tags. __Skin builders__ and others who want to change the appearance of the default installation or any of the skins can use this document to see what styles can be created for these html elements.

---++ Naming conventions

   1 All Foswiki class names have the prefix =twiki=. So: foswikiAlert, twikiToc, etcetera. Remember that CSS class names are case sensitive - Foswiki CSS uses lowercase =tw=.
   1 If you define your own CSS classes, it is preferable that you do not use the =twiki= prefix to prevent undesired overriding effects.

A wide range of standard styles are used in the Foswiki core code and topics, and more are used in plugins. The following is an exhaustive list of all styles defined by the Pattern skin. For the most part, the names are the only documentation of the purpose of the style. For more information on how these styles are used, read the code (sorry!)
